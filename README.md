# 介绍说明

​	此程序基于 vue + vue-cli + Express + mySql，后台代码放于 HTTP 文件夹内，**主要目的** 是再熟悉一下 Vue 的用法。



### 版本依赖

- vue-cli  2.9.6
- vue  2.5.2
- node 12.11.1

### 功能介绍

​	实现对“备忘录”的增、删、查，使用 better-scroll 实现页面竖直方向上的滚动
